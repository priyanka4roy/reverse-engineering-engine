# Reverse Engineering Engine

A comprehensive reverse engineering engine for analyzing large codebases, detecting dead code, identifying duplicate functionality, and performing semantic analysis using AST and embeddings.

## Components

- **hello-world-app**: Angular test application for validating the engine
- **ast-extractor**: TypeScript/AST parsing and extraction
- **ast-normalizer**: AST normalization and semantic structuring (Phase 2)

## Setup

git clone https://github.com/priyanka4roy/reverse-engineering-engine.git
